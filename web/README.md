Kafka System Control 4.0 — это эволюция проекта в сторону полноценной платформы администрирования. Новая версия добавляет веб-интерфейс, реализованный по микросервисному принципу: фронтенд на React (SPA), бэкенд на Go (REST API). Сервисы упакованы в Docker-образы и готовы к развёртыванию в Kubernetes. Ansible-плейбуки и Jenkins-пайплайны автоматизируют сборку и деплой, делая проект готовым к промышленному использованию. CLI-часть остаётся неизменной и полностью совместима с версией 3.1.
