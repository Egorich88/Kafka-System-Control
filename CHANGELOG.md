# Changelog

Все заметные изменения в этом проекте будут задокументированы в этом файле.

## [4.0] - 2026-xx-xx (планируется)
### Добавлено
- Веб‑интерфейс на React + Go (папка `web/`)
- Управление Kafka через браузер
- Поддержка нескольких окружений (DEV, PREPROD, PROD)

## [3.1] - 2026-03-14
### Добавлено
- Управление оффсетами групп (сброс на начало/конец/сдвиг/дату/конкретный оффсет) с предварительным просмотром.
- Java-утилита для поиска сообщений по ключу (`java/lib/kafka-search.jar`).
- Единый стиль оформления всех меню с ASCII-логотипом и навигацией стрелками.
- Возможность выхода из любого меню по клавише `q`.

### Изменено
- Полностью переработаны модули `describe.sh`, `create.sh`, `delete.sh` – теперь они используют `draw_module_logo` и стрелочную навигацию.
- Обновлена структура проекта (добавлена папка `java/`).

## [3.0] - 2026-02-xx
### Добавлено
- Модуль создания (create) с поддержкой ACL.
- Модуль удаления (delete).
- Логирование операций.

## [2.0] - 2023-06
### Добавлено
- Первая версия проекта с модулями describe, create, delete.
- Поддержка WSL.
